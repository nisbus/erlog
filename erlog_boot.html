Description

This little beauty allows you to start Erlang with the Erlog shell running and still has ^G and user_drv enabled. Use it as follows:

erl -noshell -noinput -s erlog_boot start

NOTE order of commands important, must be -noshell -noinput! Add -pa to find modules if necessary.

Thanks to Attila Babo for showing me how to do this.
